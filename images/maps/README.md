# 地図画像の取得方法

各スタジオの地図画像を取得する手順:

## 手順

1. Google Mapsを開く: https://www.google.com/maps
2. スタジオの住所を検索
3. 適切なズームレベル（16-17）に調整
4. スクリーンショットを撮影（推奨サイズ: 600x300px）
5. このフォルダに保存

## 推奨設定

- ズームレベル: 16-17
- サイズ: 600x300px または 1200x600px（高解像度）
- フォーマット: JPG または PNG
- ピンマーカー: 赤色で目立つように
